# mm_python_mail
用python搭的邮件服务器
## 一、环境准备
python 3.7
邮件服务器：hMailServer
  具体操作：添加一个域名mm.server.com，并在这个域名下添加amy@mm.server.com用户和user@mm.server.com用户
## 二、功能
1、收信
2、发信
3、删除邮件
4、清空邮箱
5、上传附件
